Здесь появятся логин/регистрация через Keycloak и IAM-настройки.

  • Фронтенд: Angular 21 (standalone, router)
  • Бэкенд IAM: .NET 10, модуль IAM
  • Аутентификация: Keycloak (OIDC), будет подключён позже

Проверка окружения

  1. Запусти npm start или ng serve.
  2. Открой http://localhost:4200 — должен быть этот экран.
  3. Собери фронт: npm run build.
  4. Подними Docker: cd ops && docker compose up -d web.
  5. Снова открой http://localhost:4200 — этот экран уже из контейнера.
